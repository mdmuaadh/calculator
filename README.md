
A simple, interactive web calculator with a clean tabbed interface featuring basic arithmetic operations and square root calculations. Built with pure HTML, CSS, and JavaScript for fast, responsive performance.

- **Basic Operations**: Addition, subtraction, multiplication, and division with error handling for division by zero
- **Square Root Calculator**: Dedicated tab for calculating square roots with negative number validation
- **Modern UI**: Clean design with smooth transitions, hover effects, and responsive layout
- **Tabbed Interface**: Easy switching between basic calculator and square root functions


1. Open `index.html` in a web browser
2. Use the "Basic" tab for arithmetic operations or "Square Root" tab for square root calculations
3. Enter numbers and click the operation buttons to see results
- HTML5
- CSS3 (modern styling with transitions)
- Vanilla JavaScript (no external dependencies)
